Version 2.0.0


The 2.0 update to Youtube Tabs has finally arrived!

In this update, the codebase and UI have been completely overhauled. The extension has been almost completely rebuilt from the ground up in order to make it more maintainable, reliable, and efficient. The UI has also been given a thorough makeover and redesign to make it easier and more pleasant to use (and less scalding to look at).

Since the purpose of this update was to rebuild the extension and overhaul it, there are only few new features.



Quick-Menu for New Subscriptions


The quick-menu allows you to immediately organize a new subscription, instead of having to search for it in the subscriptions panel.



Favorite Badges


You can now favorite any subscription badge to have it sorted first.
